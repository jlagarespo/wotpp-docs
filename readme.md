# The wot++ documentation system
This repo contains the wot++ documentation system, written (obviously)
in wot++. Enjoy!
## Requirements
* [wot++](https://github.com/wotpp/wotpp)
* [GNU make](https://www.gnu.org/software/make/)
* [pygments](https://pygments.org/)
